# Get Detail Struktur

To get the data in the specified collection. If no parameters are passed, it lists all of them.
Otherwise, it lists the documents in the collection matching the specified parameters:

**Method :** <Badge>GET</Badge>

**URL :** <Badge variant="dark">/struktur/(ID)</Badge>

**Response :** <BadgeOutline variant="success">200 OK</BadgeOutline>

```json
{
  "id": 1,
  "nik": "123",
  "nip": "123",
  "name": "Kepala Desa",
  "alamat": "Alamat",
  "photo": "http://localhost:8000/struktur/photo/photo-G5mz4vq.jpeg",
  "phone": "123",
  "status": 1,
  "jabatan_id": 2,
  "deleted_at": null,
  "created_at": "2022-03-10T07:53:03.000000Z",
  "updated_at": "2022-03-10T07:53:03.000000Z",
  "jabatan": {
    "id": 2,
    "jabatan": "Kepala Desa",
    "created_at": "2022-03-10T07:51:39.000000Z",
    "updated_at": "2022-03-10T07:51:39.000000Z"
  }
}
```

**Response :** <BadgeOutline variant="warning">404 NOT FOUND</BadgeOutline>

```json
Not Found
```

**Response :** <BadgeOutline variant="danger">500 ERROR</BadgeOutline>

```json
Internal Server Error
```
