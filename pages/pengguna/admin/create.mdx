# Create Admin

To create the data in the specified collection with the specified parameters.
When a JSON object is saved, the object properties are automatically added to the collection you specified.

**Method :** <Badge variant="success">POST</Badge>

**URL :** <Badge variant="dark">/admin/register</Badge>

**Body :** <Badge variant="purple">JSON</Badge>

```json
{
  "username": "username",
  "password": "password",
  "pejabat_id": 1
}
```

**Response :** <BadgeOutline variant="success">200 OK</BadgeOutline>

```json
{
  "message": "success add admin",
  "status": 200
}
```

**Response :** <BadgeOutline variant="info">422 UNPROCESSABLE CONTENT</BadgeOutline>

```json
{
  "error": {
    "username": ["Username tidak boleh kosong"],
    "password": ["Password tidak boleh kosong"],
    "pejabat_id": ["Id tidak boleh kosong"]
  }
}
```

**Response :** <BadgeOutline variant="danger">500 ERROR</BadgeOutline>

```json
Internal Server Error
```
