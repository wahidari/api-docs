# Update Penduduk

To edit the data in the specified collection with the specified parameters.
When a JSON object is saved, the object properties are automatically added to the collection you specified.

**Method :** <Badge variant="success">POST</Badge>

**URL :** <Badge variant="dark">/penduduk/(ID)?_method=PUT</Badge>

**Body :** <Badge variant="purple">Form Data</Badge>

| Field               | Type    | Required |
| :----------         | :------ | :------: |
| nama                | String  |    ✔️    |
| nik                 | String  |    ✔️    |
| no_kk               | string  |    ✔️    |
| sex                 | string  |    ✔️    |
| tempatlahir         | string  |    ✔️    |
| tanggallahir        | string  |    ✔️    |
| alamat              | string  |           |
| nama_ayah           | string  |           |
| nama_ibu            | string  |           |
| hubungan            | string  |    ✔️    |
| agama               | string  |    ✔️    |
| pendidikan_terakhir | string  |    ✔️    |
| kawin               | string  |    ✔️    |
| kewarganegaraan     | string  |    ✔️    |
| pekerjaan           | string  |    ✔️    |
| dusun               | string  |    ✔️    |
| rw                  | string  |    ✔️    |
| rt                  | string  |    ✔️    |

**Response :** <BadgeOutline variant="success">200 OK</BadgeOutline>

```json
{
  "message": "Success add penduduk",
  "status": 200
}
```
