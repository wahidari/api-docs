# Update Kelompok

To edit the data in the specified collection with the specified parameters.
When a JSON object is saved, the object properties are automatically added to the collection you specified.

**Method :** <Badge variant="info">PUT</Badge>

**URL :** <Badge variant="dark">/kelompok/(ID)</Badge>

**Body :** <Badge variant="purple">JSON</Badge>

```json
{
  "nama_kelompok": "Kelompok C",
  "mr_kelompok_id": "26",
  "ketua_id": "1",
  "dusun_id": "1",
  "rw_id": "1",
  "rt_id": "1"
}
```

**Response :** <BadgeOutline variant="success">200 OK</BadgeOutline>

```json
{
  "message": "Success update kelompok",
  "status": 200
}
```

**Response :** <BadgeOutline variant="info">422 UNPROCESSABLE CONTENT</BadgeOutline>

```json
{
  "error": {
    "nama_kelompok": ["Nama Kelompok Tidak boleh kosong"],
    "mr_kelompok_id": ["MR Kelompok ID Tidak boleh kosong"],
    "ketua_id": ["Ketua ID Tidak boleh kosong"],
    "dusun_id": ["Dusun ID Tidak boleh kosong"],
    "rw_id": ["RW ID Tidak boleh kosong"],
    "rt_id": ["RT ID Tidak boleh kosong"]
  }
}
```

**Response :** <BadgeOutline variant="danger">500 ERROR</BadgeOutline>

```json
Internal Server Error
```
