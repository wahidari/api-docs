# Edit Profil

To edit the data in the specified collection with the specified id.
When a JSON object is saved, the object properties are automatically added to the collection you specified.

**Method :** <Badge variant="info">PUT</Badge>

**URL :** <Badge variant="dark">/profil</Badge>

**Body :** <Badge variant="warning">JSON Data</Badge>

```json
{
  "nama_desa": "Nama Desa",
  "kode_desa": "Kode Desa",
  "kode_pos": "Kode Pos",
  "alamat": "Alamat Kantor",
  "email_desa": "Email",
  "phone_kantor": "Telepon Kantor",
  "website": "Website",
  "phone_polisi": "Telepon Polisi",
  "phone_pemadam": "Telepon Pemadam",
  "phone_pln": "Telepon PLN",
  "nama_kecamatan": "Nama Kecamatan",
  "kode_kecamatan": "Kode Kecamatan",
  "nama_kepala_camat": "Nama Camat",
  "nip_kepala_camat": "NIP Camat",
  "nama_kabupaten": "Nama Kabupaten",
  "kode_kabupaten": "Kode Kabupaten",
  "nama_provinsi": "Nama Provinsi",
  "kode_provinsi": "Kode Provinsi"
}
```

**Response :** <BadgeOutline variant="success">200 OK</BadgeOutline>

```json
{
  "message": "Success edit profil",
  "status": 200
}
```
