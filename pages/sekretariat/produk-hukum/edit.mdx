# Edit Produk Hukum

To edit the data in the specified collection with the specified id.
When a JSON object is saved, the object properties are automatically added to the collection you specified.

**Method :** <Badge variant="success">POST</Badge>

**URL :** <Badge variant="dark">/produk-hukum/(ID)</Badge>

**Body :** <Badge variant="warning">Form Data</Badge>

| Field      |   Type   | Required |
| :----------| :------- | :------: |
| produk     | String   |    ✔️   |
| tahun      | String   |    ✔️   |
| attachment | File/PDF |    ✔️   |

**Response :** <BadgeOutline variant="success">200 OK</BadgeOutline>

```json
{
  "message": "Success edit produk hukum",
  "status": 200
}
```

**Response :** <BadgeOutline variant="info">422 UNPROCESSABLE CONTENT</BadgeOutline>

```json
{
  "error": {
    "produk": ["Nama produk hukum tidak boleh kosong"],
    "tahun": ["Tahun produk hukum tidak boleh kosong"],
    "attachment": ["Attachment tidak boleh kosong"]
  }
}
```

**Response :** <BadgeOutline variant="warning">404 NOT FOUND</BadgeOutline>

```json
Not Found
```

**Response :** <BadgeOutline variant="danger">500 ERROR</BadgeOutline>

```json
Internal Server Error
```
