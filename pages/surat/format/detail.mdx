# Get Detail Format Surat

To get the data in the specified collection. If no parameters are passed, it lists all of them.
Otherwise, it lists the documents in the collection matching the specified parameters:

**Method :** <Badge>GET</Badge>

**URL :** <Badge variant="dark">/surat/format/(ID)</Badge>

**Response :** <BadgeOutline variant="success">200 OK</BadgeOutline>

```json
{
  "id": 1,
  "kode": "S-1",
  "nama_surat": "Surat Keterangan",
  "pemohon": 1,
  "format_file": "http://localhost:8000/surat/format/file/surat-keterangan-WAv4pWq.rtf",
  "format_field": "[{\"type\":\"penduduk\",\"column\":\"nama\",\"title\":\"Nama Penduduk\",\"format\":\"napend\"},{\"type\":\"penduduk\",\"column\":\"alamat\",\"title\":\"Alamat Penduduk\",\"format\":\"alamat\"}]",
  "requirement": "[1,2]",
  "perangkat_id": 1,
  "deleted_at": null,
  "created_at": "2022-03-11T11:01:13.000000Z",
  "updated_at": "2022-03-11T11:01:13.000000Z"
}
```

**Response :** <BadgeOutline variant="warning">404 NOT FOUND</BadgeOutline>

```json
Not Found
```

**Response :** <BadgeOutline variant="danger">500 ERROR</BadgeOutline>

```json
Internal Server Error
```
